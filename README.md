# 🌾 Agricultura Mágica com Tema Harry Potter ✨

## Bem-vindo!

Este é um site educativo e interativo sobre agricultura, mas com um toque mágico do universo de Harry Potter! O projeto combina curiosidades, dados e estatísticas sobre agricultura com elementos fantásticos da série.

## 🎯 O que você encontrará aqui

### 📚 Seções Principais

1. **Início (Home)** - Introdução à Agricultura Mágica
2. **Curiosidades** - Fatos interessantes sobre plantas mágicas
3. **Dados e Estatísticas** - Comparação entre agricultura tradicional e mágica
4. **Feitiços Essenciais** - Feitiços utilizados em agricultura
5. **Plantas Mágicas** - Catálogo de plantas cultivadas

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla

## 🎨 Características Especiais

✨ Design imersivo com tema Harry Potter
📱 Responsividade total
🔮 Interatividades e efeitos especiais

## 📊 Dados Principais

- **Produtividade**: 350% com feitiços de crescimento
- **Tempo de Colheita**: 72h com magia vs 120 dias tradicional
- **Taxa de Sucesso**: 89% com proteção mágica
- **Economia de Água**: 60% com feitiços climáticos

## 🪄 Como Usar

1. Abra o arquivo `index.html` em um navegador
2. Navegue pelas seções usando o menu
3. Descubra os efeitos interativos!
4. **Dica**: Clique 5 vezes no título para desbloquear mensagens especiais!
